# 📖 ターゲット1900 - フラッシュカード学習アプリ

**ターゲット1900**の1900語をフラッシュカード形式で効率的に学習できるWebアプリケーションです。

## ✨ 特徴

- 📱 **レスポンシブ設計** - パソコン、タブレット、スマートフォンで快適に学習
- 🌓 **ダークモード対応** - システム設定に自動対応
- 📊 **SRS（間隔反復）システム** - 科学的に単語を復習
- 💾 **ローカル保存** - ブラウザのストレージに学習記録を自動保存
- 🎯 **範囲設定機能** - 学習したい単語の範囲を自由に設定
- 🔍 **単語検索** - 単語・意味で素早く検索
- ⚡ **軽量・高速** - 外部依存なしの100%HTML/CSS/JavaScript

## 🚀 使い方

### 1. サイトにアクセス
ブラウザでこのリポジトリのPages にアクセスします。

### 2. 学習ビュー
- カードをタップして答えを確認
- 難易度を4段階で評価
  - **もう一度**：すぐに出題
  - **難しい**：2〜3日後に出題
  - **良い**：7日後に出題
  - **簡単**：14日後に出題

### 3. 範囲設定
- 学習する単語の番号範囲を指定
- よく使う範囲（1〜100、301〜600など）はワンクリック設定

### 4. 単語一覧
- 全1900語の一覧表示
- 単語や意味で検索
- 各単語の学習状態を確認（新規、今日、日数）

## 📊 統計情報

トップ画面に表示される4つの統計：
- **今日の予定** - 本日復習予定の単語数
- **完了** - セッションで完了した単語数
- **未学習** - 学習範囲内の未学習単語数
- **範囲内** - 現在の学習範囲内の総単語数

## 💾 データについて

- すべての学習記録はブラウザのLocalStorageに保存されます
- クラウド同期はなく、デバイス固有の記録となります
- キャッシュクリアでリセットできます

## 🔧 技術仕様

### 学習アルゴリズム
SM-2アルゴリズムを採用した間隔反復システム：
- `ef`（Ease Factor）：難易度係数（初期値: 2.5）
- `interval`：次の出題までの日数
- `rep`：反復回数
- `due`：次の出題予定日時

### 評価スコア
- **0点**：もう一度（即座に再出題）
- **1点**：難しい（復習間隔を縮小）
- **2点**：良い（通常の復習スケジュール）
- **3点**：簡単（復習間隔を拡大）

## 📝 単語データ構造

各単語は以下の形式で保存されています：
```javascript
[番号, "単語", "品詞", "意味"]
```

例：
```javascript
[1, "create", "動詞", "を創り出す；を引き起こす"]
```

## 🎨 カスタマイズ

CSSカラー変数を変更することでテーマをカスタマイズできます：

```css
:root {
  --bg: #ffffff;
  --text: #1a1a1a;
  --blue: #2563eb;
}
```
